### 一、项目功能分析文档

1. 首页:值班热线、安全小助手、思政热线、今日头条、热门活动、反诈之星、二级学院排行
2. 社区:反诈行动、心得体会、反诈资讯
3. 安全学堂:反诈案例、视频、视觉探索
4. 我的:签到、学习积分、积分可视化、我的(收藏、评论、活动、心得)、积分兑换、发布活动、活动审核、账户管理
5. 活动发布页面
